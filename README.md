## 🚀 Sobre o Projeto

O **Contador de Palavras** foi desenvolvido com o objetivo de praticar lógica de programação e aplicar estrutura profissional desde projetos pequenos.

O sistema:

* Recebe uma frase digitada pelo usuário
* Valida o tipo da entrada
* Conta automaticamente o número de palavras
* Ajusta corretamente a pluralização
* Organiza a execução através da função `main()`
* Utiliza o padrão `if __name__ == "__main__"` para controle de execução
